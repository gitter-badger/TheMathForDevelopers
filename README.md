### 이 글을 읽을 때 주의하실 점.

- 해당 과정들은 정확히 당식의 직무에 일치하는 스킬셋이 아닙니다.
- 업무특성상 다를 수 있으며 그것을 최대한 객관화 시킨 리스트입니다.

- 이 글은 계속적으로 업데이트 되고있습니다.
- 자신의 직무에 따른 부분이 빠져있다면 건의해주세요.

- 이 글은 참여형태로 진행됩니다, 직접 `fork`하시고 pull request를 요청하세요.
